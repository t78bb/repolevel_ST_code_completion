This is the known requirement information for the function to be completed:
This code is intended to implement a function that converts axis status information into a corresponding descriptive string based on priority order.
This provides you with a plan of implementation steps for the function to be completed. Subsequent ST code must strictly follow this plan and must not deviate from the core logic. The following is the plan of implementation steps:
功能规划:
1. 检查 fbReadStatus 是否处于有效状态（启用状态 Enable = TRUE）。
2. 根据 fbReadStatus 的 Done 属性，优先判断轴的是否运动完成，并返回对应的描述字符串。
3. 如果 MotionActive 属性为 TRUE，则返回轴运动中的描述字符串。
4. 如果 ErrorActive 属性为 TRUE，则返回轴错误状态的描述字符串。
5. 如果轴处于其他明确的状态（如 StandStill）时，根据状态标识返回对应的描述字符串。
6. 对于未定义或无匹配状态的情况，返回默认的未知状态描述字符串。
